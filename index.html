Chantilly Robotics Code for 2013 FRC Challenge

This project contains all of FIRST Team 612 Chantilly Robotics code for the 2013 FRC Challenge.

Usage

For now, this project assumes that you already have ucpp_ installed and configured for your system. The build process is as follows:

$ ucpp init #run once $ ucpp configure #run each time the files in the project have changed $ make #compile and link all code (debug) $ make DEBUG_MODE=0 #compile and link all code (release) $ make deploy #deploy code to robot

For more information regarding ucpp, see 'ucpp.rst'.

Configuring ++++++++++++++++

  • When imaging it is necessary to select the option for WindRiver C++, and please enable NetConsole (except for in perhaps release builds).
  • Remember that the cRIO runs an ftp server that allows anonymous ftp access.
  • The robot needs to be imaged with version 43 of the firmware to be legal.
